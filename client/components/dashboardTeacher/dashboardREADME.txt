# 📁 Dashboard Structure - Структура дашборда

## 🎯 **Обзор**
Дашборд разделен на логические компоненты для удобства разработки и поддержки.

## 📂 **Структура файлов**

```
src/
├── components/
│   ├── dashboard/                    # 📁 Компоненты дашборда
│   │   ├── DashboardHeader.jsx       # 🔝 Верхний хедер
│   │   ├── DashboardSidebar.jsx      # 📋 Боковое меню
│   │   ├── ProfileSection.jsx       # 👤 Секция профиля
│   │   ├── StudySection.jsx         # 📚 Секция курсов
│   │   ├── ChatSection.jsx          # 💬 Секция чата
│   │   └── NotificationBanner.jsx   # 🔔 Уведомления
│   ├── courses/                     # 📚 Компоненты курсов
│   │   └── CourseDetail.jsx         # 📖 Детали курса
│   ├── chat/                        # 💬 Компоненты чата
│   │   └── Chat.jsx                 # 🗨️ Основной чат
│   └── dashboardTeacher/            # 👨‍🏫 Главный дашборд
│       └── dashboardTeachers.jsx    # 🏠 Основной компонент
```

## 🧩 **Компоненты и их функции**

### 🔝 **DashboardHeader.jsx**
**Назначение:** Верхний фиксированный хедер
**Содержит:**
- 🅰️ Логотип приложения
- 🌍 Переключатель языков (RU/EN/KZ)
- 🚪 Кнопка выхода

**Props:**
```javascript
{
  language: string,           // Текущий язык
  onLanguageChange: function, // Обработчик смены языка
  onLogout: function,         // Обработчик выхода
  translations: object        // Объект переводов
}
```

### 📋 **DashboardSidebar.jsx**
**Назначение:** Боковое навигационное меню
**Содержит:**
- 📝 Пункты меню (Профиль, Учеба, Чат)
- ✨ Активные состояния кнопок
- 📱 Адаптивность для мобильных

**Props:**
```javascript
{
  activeSection: string,      // Активная секция
  onMenuClick: function,      // Обработчик клика по меню
  isMobile: boolean,          // Флаг мобильной версии
  translations: object,       // Объект переводов
  language: string           // Текущий язык
}
```

### 👤 **ProfileSection.jsx**
**Назначение:** Форма редактирования профиля
**Содержит:**
- 📝 Поля ввода данных пользователя
- 💾 Кнопка сохранения
- ⏳ Состояния загрузки

**Props:**
```javascript
{
  dataTabla: object,          // Данные пользователя
  isLoading: boolean,         // Состояние загрузки
  isMobile: boolean,          // Флаг мобильной версии
  translations: object,       // Объект переводов
  language: string,           // Текущий язык
  onFieldChange: function,    // Обработчик изменения полей
  onSave: function           // Обработчик сохранения
}
```

### 📚 **StudySection.jsx**
**Назначение:** Список курсов и навигация к деталям
**Содержит:**
- 🎯 Карточки курсов
- 🔍 Переход к деталям курса
- ↩️ Возврат к списку

**Props:**
```javascript
{
  selectedCourse: object,     // Выбранный курс
  onCourseSelect: function,   // Обработчик выбора курса
  onBackToCourses: function,  // Обработчик возврата
  isMobile: boolean,          // Флаг мобильной версии
  translations: object,       // Объект переводов
  language: string,           // Текущий язык
  isHovered: number,          // Индекс наведенной карточки
  onHover: function          // Обработчик наведения
}
```

### 💬 **ChatSection.jsx**
**Назначение:** Обертка для компонента чата
**Содержит:**
- 🏷️ Заголовок секции
- 💬 Компонент Chat

**Props:**
```javascript
{
  user: object,               // Данные пользователя
  dataTabla: object,          // Дополнительные данные
  isMobile: boolean,          // Флаг мобильной версии
  translations: object,       // Объект переводов
  language: string           // Текущий язык
}
```

### 🔔 **NotificationBanner.jsx**
**Назначение:** Уведомления для пользователя
**Содержит:**
- ✅ Сообщение об успехе
- ❌ Кнопка закрытия
- 🎬 Анимация появления

**Props:**
```javascript
{
  show: boolean,              // Показать/скрыть
  message: string,            // Текст уведомления
  onClose: function          // Обработчик закрытия
}
```

## 🏠 **Главный компонент - dashboardTeachers.jsx**
**Назначение:** Координирует все компоненты
**Управляет:**
- 🔄 Состоянием приложения
- 📡 API запросами
- 🎛️ Логикой навигации
- 🌍 Переводами

## 🎨 **Преимущества модульной структуры**

### ✅ **Удобство разработки:**
- 🔧 Каждый компонент отвечает за одну задачу
- 🐛 Легче находить и исправлять ошибки
- 🔄 Простое переиспользование кода
- 👥 Возможность параллельной разработки

### ✅ **Легкость поддержки:**
- 📝 Изменения в одном месте
- 🧪 Простое тестирование отдельных частей
- 📚 Понятная структура для новых разработчиков
- 🔍 Быстрый поиск нужного функционала

### ✅ **Масштабируемость:**
- ➕ Легко добавлять новые секции
- 🔧 Простая модификация существующих
- 🎨 Независимые стили для каждого компонента
- 📱 Адаптивность на уровне компонентов

## 🚀 **Как вносить изменения**

### 📝 **Изменить хедер:**
Редактируйте `DashboardHeader.jsx`

### 📋 **Добавить пункт меню:**
Обновите `DashboardSidebar.jsx` и добавьте новую секцию

### 👤 **Изменить форму профиля:**
Редактируйте `ProfileSection.jsx`

### 📚 **Модифицировать курсы:**
Обновите `StudySection.jsx` или `CourseDetail.jsx`

### 💬 **Настроить чат:**
Измените `ChatSection.jsx` или основной `Chat.jsx`

### 🔔 **Кастомизировать уведомления:**
Редактируйте `NotificationBanner.jsx`

## 🛠️ **Пример добавления новой секции**

1. **Создать компонент** `NewSection.jsx`
2. **Добавить в сайдбар** новый пункт меню
3. **Импортировать** в главный дашборд
4. **Добавить в renderContent()** новый case
5. **Обновить переводы** для всех языков

Теперь ваш дашборд полностью модульный и готов к масштабированию! 🎉
